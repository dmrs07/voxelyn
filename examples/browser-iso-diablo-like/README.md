# Iso Diablo-like Dungeon Example

An atmospheric dungeon scene inspired by classic isometric ARPGs like Diablo.

## Features

- **Stone floor tiles** with weathering, mortar lines, and cracks
- **Sandstone brick walls** with proper isometric depth and lighting
- **Animated torches** with flickering flames positioned along walls
- **Glowing cyan runes** scattered across the floor with pulsing animation
- **Magic circle** at room center with rotating glow effects
- **Portal archways** at cardinal directions with cyan glow
- **Debris and moss** scattered near walls for detail
- **Stairs** leading down from the entrance

## Color Palette

- Warm brown/tan tones for stone and walls
- Cyan/teal for magical elements (runes, portals)
- Orange/yellow for torch flames
- Dark atmospheric background

## Running

Open `index.html` via a local server:

```bash
npx http-server -c-1
# Navigate to examples/browser-iso-diablo-like/
```

## Controls

Currently view-only. The scene animates automatically with:
- Flickering torch flames
- Pulsing rune glow
- Rotating magic circle effects
